Build
-----

In the repository directory, i.e. 'C:/dev/travis_ci_test".

	mkdir build
	cd build
	cmake ../ -G"MinGW Makefiles" -DCMAKE_SH="CMAKE_SH-NOTFOUND"
	mingw32-make
